Pascal.

Программа состоит из двух блоков:
1. Описание переменных
2. Дейслтвия с переменными

Типы переменных:
Тип   -   байт - Диапазон 
byte      1      0..255
shortint  1      -128..127
word      2      0..65535
integer   2      -32768..32767
longint   4      -2147483648..2147483647
real      6      2,9x10^-39 - 1,7x10^38
boolean   1      true / false
char      1      ASCII (все символы)

Описание переменных происходит командой var.
Пример: var a:integer;
	    b:longint;

Во второй части вы производите все операции с вышеперечисленными переменными.
Вторая часть в программе выделяется особыми словами begin, end. .
Пример. begin
	   command
	end. 

Действия с переменными.
Сложение (+)
Умножение (*)
Вычитание (-)
Деление:
	Нацело (div)
	Остаток (mod)
	Стандартное (/) 
Присваивание (:=)
	
Справочник команд.
assign(input, 'input.txt'); - Связать. Связывает переменную input с файлом input.txt
begin - Начать. Начинает исполнение программы / действий в цикле
clrscr; - Отчистить экран. Необходимо для консольных приложений
end; - Завершить. Выполнение действий окончено (для циклов)
end. - Завершить. Выполнение программы окончено
for x1 to x2 do ... - Цикл "ОТ-ДО". Выполнит действия x2-x1 раз  
if ... then ... else ... - Условный оператор. Если условие = true тогда действия then, иначе else
read(a); - Считать. Получает данные из input (файл / устройство ввода) и присвоить их переменной a
readln(a); - Считать, далее с новой строки.
reset(input); - Открыть для чтения. Открывает файл из переменной input для чтения
rewrite(output); - Открыть для записи. Открывает файл из переменной output для записи
var a:integer; - Описать переменную. Описывает a как integer
while ... do ... - Цикл "пока". Выполняет команды пока условие = true
write(a); - Вывести. Выводит данные в output (файл / устройство вывода)
writeln(a); - Вывести, далее с новой строки.
