Самолет взлетает в X (целое, 0<=X<=23) часов по местному времени в часовом поясе номер M (целое, 0<=M<=23). После полета в течение K (целое, 1<=K<=12) часов он приземляется в часовом поясе номер N (целое, 0<=N<=23). Определите местное время в пункте приземления. Считать, что часовые пояса нумеруются с запада на восток.

Формат входных данных:
В первой строке текстового файла через пробел записаны числа X, M, K, N.

Формат выходных данных:
В выходной файл требуется выдать местное время в пункте назначения в часах.

В данной задаче есть три случая полета:
1. Пояс вылета больше пояса прилета ( m > n )
2. Пояс вылета меньше пояса прилета ( m < n )
3. Пояса вылета и прилета равны     ( m = n )
te = разница во времени поясов

Всегда время вычисляется по формуле: (x + k + te) mod 24

НЕ ЗАБУДТЕ:
- Самолет выбирает кротчайший путь!
