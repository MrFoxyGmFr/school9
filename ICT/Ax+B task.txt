Для заданных целых чисел A и B вывести двучлен Ax+B в корректном виде. Для A, B=0  результатом должен быть 0. 

Формат файла входных данных:
Во входном файле заданы два целых числа не превосходящих 32000.

Формат файла выходных данных:
Вывести математически корректное выражение. 

у нас 4 случая на а:
 a = 1      - Вывести а
 a = -1     - вывести -а
 a = 0      - Ничего не выводить (по а)
 -1 < a < 1 - Вывести саму переменную(тк. знак хранится в переменной)

И так же 4 сучая:
  b = 0 и а = 0   - 0
  b = 0, а <> 0   - ничего не выводить
  b положительное - "+", b
  b отрицательное - b

Решение пожалуйста напишите сами)

НЕ ЗАБУДТЕ:
- Write не делает пробелов и строку не переносит, после завершения.
- Числа могут быт как положительными, так и отрицательными.
