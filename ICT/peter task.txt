На доске написано N натуральных чисел. Петя выбирает два из них и, если они одинаковые, одно из них стирает. Если числа разные, то он вычитает из большего числа меньшее до тех пор, пока они не станут одинаковыми. В конце концов на доске осталось одно число. Какое?

Формат файла входных данных:
Первая строка входного файла содержит натуральное число N.
На второй строке находятся N натуральных чисел, разделённых пробелом. Все числа longint.

Формат файла выходных данных:
Все возможные числа, которые может получить Петя, через пробел.

То, что делает Петя - это находит НОД.

Например: 6 9 3 54 - ряд из четырех чисел (Проведем действия Пети)
1. Из 9 - 6 = 3
2. Из 6 - 3 = 3
3. 3 == 3 значит одно стирает.
4. 3 == 3 значит одно стирает.
(Промежуточный итог - ряд 3 54 - осталось 2 числа)
5. Из 54 - 3 = 51 итд до 6 - 3 = 3
6. 3 == 3 значит одно стирает.
Итог - одно число 3.

Это и был НОД. Ну а найти НОД х чисел не сложно.

НЕ ЗАБУДТЕ:
- По условию все чилса - Longint!
- Вывод - последнее число (НОД)
